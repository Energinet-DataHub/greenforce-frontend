# Date Format

Watt ships with an Angular pipe utility for easy formatting of dates. It supports outputting
two different formats, date with time and date without time. The format is always in Danish
locale and in the Europe/Copenhagen timezone. The default format is date without time.

The pipe accepts strings or a DateRange object of strings in ISO 8601 format.

## Example

The following code shows the different use cases of the WattDatePipe:

```ts
import { Component } from '@angular/core';
import { WattDatePipe } from '@energinet-datahub/watt/utils/date';

@Component({
  standalone: true,
  imports: [WattDatePipe],
  template: `
    <!-- 01-01-2022 -->
    <span>{{ date | wattDate }}</span>
    <!-- 01-01-2022 00:00 -->
    <span>{{ date | wattDate : 'long' }}</span>
    <!-- 25-03-2019 ― 27-03-2019 -->
    <span>{{ dateRange | wattDate }}</span>
    <!-- 25-03-2019 23:00 ― 27-03-2019 22:59 -->
    <span>{{ dateRange | wattDate : 'long' }}</span>
  `,
})
class MyComponent {
  date = '2021-12-31T23:00:00Z';
  dateRange = {
    start: '2019-03-25T22:00:00Z',
    end: '2019-03-27T21:59:59Z',
  };
}
```
