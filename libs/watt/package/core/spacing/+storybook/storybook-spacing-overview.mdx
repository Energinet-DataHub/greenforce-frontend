# How to use

## CSS Custom Variables

```css
--watt-space-xs
--watt-space-s
--watt-space-m
--watt-space-l
--watt-space-xl
```

### SCSS Mixins

```scss
@use '@energinet-datahub/watt/utils' as watt;

// Inset
@include watt.space-inset-xs;
@include watt.space-inset-s;
@include watt.space-inset-m;
@include watt.space-inset-l;
@include watt.space-inset-xl;

// Inset Squish
@include watt.space-inset-squish-xs;
@include watt.space-inset-squish-s;
@include watt.space-inset-squish-m;
@include watt.space-inset-squish-l;

// Inset Stretch
@include watt.space-inset-stretch-s;
@include watt.space-inset-stretch-m;

// Stack
@include watt.space-stack-xxs;
@include watt.space-stack-xs;
@include watt.space-stack-s;
@include watt.space-stack-m;
@include watt.space-stack-l;
@include watt.space-stack-xl;

// Inline
@include watt.space-inline-xxs;
@include watt.space-inline-xs;
@include watt.space-inline-s;
@include watt.space-inline-m;
@include watt.space-inline-l;
@include watt.space-inline-xl;
```
