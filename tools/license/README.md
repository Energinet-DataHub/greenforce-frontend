# License

To add license header (where missing) to all files in the project, run the following command:

```sh
bun nx g @energinet-datahub/tools/license:add --dryRun=true
```

To remove license header from certain files in the project, run the following command:

```sh
bun nx g @energinet-datahub/tools/license:remove --pattern="**/*.ts" --dryRun=true
```
